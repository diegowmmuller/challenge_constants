Este projeto é um exercício introdutório em C++ utilizando CMake como sistema de configuração de build e Ninja como executor de compilação.
O objetivo é praticar o uso de constantes, tipos primitivos e operações básicas, além de compreender o fluxo de build moderno em C++ (configuração, compilação e linkedição).

O projeto foi desenvolvido no CLion em ambiente Linux, seguindo boas práticas como:

Separação entre código-fonte e artefatos de build

Uso de CMake para portabilidade do processo de compilação

Versionamento apenas dos arquivos essenciais ao projeto
